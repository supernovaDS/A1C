## ⚡ Features

ConvertAll provides a vast array of utilities that execute heavily on the client side, ensuring that your data stays secure.

### 📄 PDF Tools
- **Image to PDF**: Create high-quality PDFs from standard JPG/PNG images.
- **Word to PDF**: Accurate document reproduction from DOCX formatting.
- **PDF to Images**: Extract pages in high-resolution JPG/PNG formats inside a ZIP.
- **Merge PDFs**: Combine multiple PDFs through an interactive drag-and-drop interface.
- **Compress PDF**: Drastically reduce your file size without compromising readability.
- **Split PDF**: Visual page extraction and burst-to-single-page functionalities.
- **Rotate PDF**: Permanently rewrite PDF metadata to correct orientation.
- **Protect/Unlock PDF**: Military-grade local encryption tools.
- **Watermark PDF**: Apply transparent text or image overlays natively.
- **PPT/Excel to PDF**: Transform presentations and spreadsheets into static PDF documents.

### 🖼️ Image Tools
- **Image Converter**: Robust batch conversion across JPG, PNG, and WebP, featuring visual compression size estimation and parallel processing.

### 🧠 AI & Document Intelligence
- **Document AI**: Advanced OCR (Optical Character Recognition) to extract text or intelligently summarize documents quickly.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Routing**: React Router DOM
- **PDF Manipulation**: [pdf-lib](https://pdf-lib.js.org/), [pdfjs-dist](https://mozilla.github.io/pdf.js/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **File Utilities**: `jszip`, `file-saver`, `react-dropzone`

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn

### Installation
1. Clone the repository
2. Navigate into the client directory:
   ```sh
   cd client/AllinOneConverter
   ```
3. Install NPM packages:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```

## 🎨 Design Principles
The UI follows a stringent "Utility-First" professional approach:
- **Focus on Legibility:** Inter typography, optimized hierarchy.
- **No Flashy Elements:** Avoidance of heavily glowing shadows, maintaining a clean slate, blue, and green-focused aesthetic.
- **Reliability:** Direct loading indicators, processing status outputs, and immediate file downloads.
